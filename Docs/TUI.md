### Overview
This will be a terrminal user interface api built in c++ targeting both pythong and c++. The api is mostly for my self to allow me to have a scalable easy to use way to build user interfaces inpaticuler terrminal user interfaces for any project i build in the futer. This project will also be a way to become very familer with the way that the terrminal works as well as c++ it's self. This project's goals is to, one teach me c++ and how terrminals work and, two give me a way to make tui's easialy for later projects.
### Features
As a TUI interface api this api should allow me to easly and quicly build TUI's in both python and c++. This TUI should include but not be limeted to:
- A Fast featur rich [[Rendering]] for the tui
- The abilty to interface with the image viewigng capabliys of terrminals like kitty for both image and possably video show cases
- The abilty to have fast hot key centrec work flow with in apps built by this app
- It will have many widgets to uses as well as a low level widget building api
- Markdown rendering 
### Interface
The interface for this TUI will be an api using pybind11 for python with native c++ headers for the c++ users of this libbray. The api will be like a cross between a gui api and a game engine api. I will uses pygames api as insperation for much of the desgin although i will improve upon this api for instance I will be using a diffrent input methoud then pygame uses.
